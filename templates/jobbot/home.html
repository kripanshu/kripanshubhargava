JobBot

Jobbot is a machine learning bot which I planed to make from scratch. It is a Bot which help all type of job applicants who use LinkedIn, AngelList or other platforms to search for jobs and want to keep track of these jobs. The purpose of this bot is to create and update and excel file ( maybe JSON) for you.



GitHub Repository
Project Scope
  • The project scope is to convert this python application into machine learning bot.
  • Using Rule Based Natural Programming Language concept to apply on closed vocabulary of this job world.

Project Current Status
  • Currently There is a python script which takes user credentials and use GMAIL API to access the emails and gets all the job related emails from LinkedIn and Angel List ( For Now ).
